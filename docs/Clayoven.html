    module Clayoven

    The main clayoven module

    Clayoven::Toplevel, Clayoven::Httpd, Clayoven::Init, and other modules required by them are nested under this. Matches ARGV[0] against ‘httpd’, ‘aggressive’, ‘init’; otherwise runs clayoven in the default incremental-build mode.

    The incremental build relies on the git index, so if you committed files before running clayoven, run clayoven aggressive.

    Only the aggressive variant generates the sitemap.
